# Sensitive Lexicon Integration

This directory is compatible with the text-file layout used by
`konsheng/Sensitive-lexicon`.

How to strengthen moderation further:

1. Clone or copy the upstream repository into this directory, or point
   `SENSITIVE_LEXICON_DIR` to the cloned repository root.
2. Keep the `Vocabulary/` and `Organized/` folders in plain `.txt` form.
3. The backend will reload the directory automatically on a short cache window.

Bundled files in this repo are only a minimal starter set for local development.
