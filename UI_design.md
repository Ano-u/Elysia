# Elysia 项目 UI/UX 深度设计规范 (UI_design.md)

## 1. 核心美学与主题 (Core Aesthetic)
**“无瑕、纯净、温柔、星海与记忆”**
基于《崩坏3》爱莉希雅（Elysia / 真我·人之律者）的角色概念，打造一个充满情感温度与极致浪漫的交互空间。

- **关键词**：Crystal Bloom (水晶花绽放)、Liquid Glass (液态玻璃)、Star Sea (浩瀚星海)、Elysian Signets (乐土刻印)、Calm Motion (呼吸感动效)。
- **核心材质**：毛玻璃（高斯模糊 `backdrop-filter: blur`）、微色散描边（模拟水晶折射）、柔和高光。

## 2. 色彩系统 (Color Palette)
避免使用过于沉重或刺眼的颜色，整体保持明亮、通透或深邃的梦幻感：
- **基调底色 (Base)**：
  - **星海深空 (Universe Background)**：深邃紫 (`#1A0B2E`) 渐变过渡到 星空蓝 (`#0D1B2A`)，带有动态呼吸感。
  - **常规底色 (Home/MindMap)**：低噪点冰晶白 (`#F8F9FA`) 或 晨雾蓝 (`#F0F4F8`)。
- **主题色 (Primary)**：珊瑚粉/水晶粉 (`#FFA6C9` / `#FFC0CB`) —— 代表爱与温柔。
- **点缀色 (Accent)**：
  - 浅金高光 (`#F9E596`) —— 用于强调、记忆丝线、金句节点。
  - 紫罗兰 (`#8A2BE2` 柔和版) —— 用于神秘感或深层逻辑节点。

## 3. 文案系统 (Copywriting System - 爱莉希雅语气)
完全摒弃冰冷的机器提示，所有文案需符合“粉色妖精小姐”的人设：热情、夸赞、充满爱意，句末常带音符“♪”。
- **引导输入**：“嗨，今天有什么绚丽的想法，想要告诉我吗？♪” / “把这份无瑕的记忆交给我保管吧♪”
- **状态反馈**：“嘘...这份灵感已经悄悄藏进水晶里了哦♪” / “哎呀，星海的连接似乎有些微弱，但我会一直在这里等你的♪”
- **空状态/卡顿**：“遇到烦心事了吗？不如深呼吸，让思绪像飞花一样飘散吧~ 需不需要我给你一点小灵感呢？♪”
- **互动共鸣**：“感受到这份真挚的爱了吗？♪”

## 4. 核心模块视觉与交互方案

### 4.1 星海 (Universe) - 飞鳐与光晕的交响
- **背景**：深空蓝紫色流体星云（CSS `radial-gradient` + 缓慢的 `transform/opacity` 动画）。
- **主题星团（热点）**：使用 **Canvas API** 绘制半透明的“晶体飞鳐”。飞鳐在屏幕背景层缓缓游动，尾迹洒落粉色几何粒子。
- **内容气泡**：他人的记录化作悬浮的水晶气泡。当用户拖拽视口“漫游”时，气泡从远处的模糊点（缩放+模糊）平滑过渡到清晰的液态玻璃卡片。
- **共鸣动效 (Hug/Heart)**：拖拽表情到气泡上时，触发 Canvas 粒子爆发（粉色水晶飞花），并伴随 CSS `box-shadow` 水波纹涟漪。
- **听觉通感**：不播放真实音频，交互时伴随闪烁的音符符号“♪”粒子飘散。

### 4.2 记忆织网 (MindMap) - 往世乐土的刻印
- **节点设计 (Nodes)**：
  - 情绪记录：粉色的多瓣水晶花。
  - 金句/高亮记录：浅金色的菱形星芒。
  - 普通记录：冰蓝色或紫罗兰色的多面体。
- **连线 (Edges)**：带有微弱光子流动动画的“金色记忆丝线”（Canvas 绘制贝塞尔曲线 + 粒子沿路径运动）。
- **聚类与绽放动效**：
  - 缩小/高密度时：聚合成一颗巨大的“原石水晶”。
  - 点击/聚焦展开时：模拟清脆的视觉破碎感，水晶解离成满屏的刻印节点（花朵绽放式的缓动弹出）。

## 5. 技术实现架构 (CSS3 + Canvas)
为了满足“首屏 ≤ 2.5s”的严苛性能要求：
1. **DOM / CSS3 负责**：页面布局、卡片容器、液态玻璃质感（`backdrop-filter`）、发光阴影（`box-shadow`）、背景渐变（`background: radial-gradient`）、基本的 Transform 形变。
2. **纯 Canvas API 负责**：飞鳐游动轨迹、复杂的连线流光、鼠标/拖拽触发的大量飞花粒子。作为一个透明的 `<canvas>` 层覆盖在背景或卡片之间。
3. **性能降级策略**：通过 `requestAnimationFrame` 监控帧率，若设备性能不足，自动减少 Canvas 粒子生成速率或隐藏飞鳐，但保留 CSS 的玻璃质感。
